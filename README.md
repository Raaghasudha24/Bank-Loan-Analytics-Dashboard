**Bank-Loan-Analytics-Dashboard**

An interactive Power BI dashboard analyzing 38K+ bank loan applications using Excel, Power Query, DAX, and Power BI.

**Project Overview**

The Bank Loan Analytics Dashboard is a Business Intelligence project developed to analyze a bank's historical loan portfolio and transform raw loan data into meaningful business insights.
The dashboard enables stakeholders to monitor loan performance, evaluate borrower characteristics, identify lending trends, and support data-driven lending decisions through interactive visualizations.
**
Project Objective**

The objective of this project is to:

- Analyze historical loan application data.
- Monitor key lending performance indicators.
- Identify borrower and regional lending trends.
- Evaluate loan portfolio performance.
- Support business decision-making using interactive dashboards.

**Dataset Information**

The dataset contains approximately 38,000 loan applications with attributes including:

- Loan ID
- Loan Status
- Loan Amount
- Funded Amount
- Total Payment
- Interest Rate
- Debt-to-Income Ratio (DTI)
- Annual Income
- Borrower Grade
- Loan Purpose
- Home Ownership
- Verification Status
- State
- Issue Date

**Tools & Technologies**

- Microsoft Excel
- Power Query
- Power BI
- DAX (Data Analysis Expressions)

**Project Workflow**

1. Data Understanding
2. Data Dictionary Creation
3. Data Cleaning using Power Query
4. Data Validation
5. Data Import into Power BI
6. DAX Measure Creation
7. Dashboard Development
8. Business Insight Generation

**Key Performance Indicators (KPIs)**

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average Debt-to-Income Ratio (DTI)

**Business Questions Answered**

- How many loan applications were processed?
- Which borrower grades received the highest funding?
- What are the most common loan purposes?
- Which states received the highest loan funding?
- What is the distribution of loan status?
- How do home ownership and verification status relate to the borrower portfolio?

**Key Findings**

- Over **38,000 loan applications** were analyzed.
- **Fully Paid** loans represented the largest share of the portfolio.
- **Grade B** borrowers received the highest funded amount.
- **Debt Consolidation** was the most common loan purpose.
- **California** recorded the highest funded loan amount.
- The average **Debt-to-Income Ratio (DTI)** remained within an acceptable range.

**Business Recommendations**

- Continue focusing on customer segments with strong repayment history.
- Expand lending strategies for high-performing borrower grades while maintaining effective risk controls.
- Introduce specialized financial products for customers seeking debt consolidation.
- Strengthen marketing efforts in high-performing states such as California.
- Improve borrower verification processes to reduce credit risk.
- Use DTI and borrower grades together during credit evaluation.

**Business Outcome**

The final deliverable is an interactive **Power BI Dashboard** that transforms raw loan data into actionable business insights.
The dashboard enables management to:

- Monitor loan portfolio performance
- Track borrower behavior
- Identify lending trends
- Evaluate credit risk
- Support strategic lending decisions
- Reduce manual reporting efforts through interactive visualization
